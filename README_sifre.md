## Postavljanje na GitHub Pages

1. Napravi novi GitHub repozitorijum
2. Ubaci `index.html` u root repozitorijuma
3. Idi na `Settings → Pages → Source: main branch`
4. Sačekaj 2 minuta → dobićeš link `https://[korisnik].github.io/[repo]`
5. Taj link šalješ polaznicima zajedno sa njihovom šifrom
